Wp Custom Meta Box plugin helps you to create meta box and display meta box data on frontend provided by Agile Infoways.com

== Description ==

= Premium Version =
Get Wp Custom Meta Box <a href="http://www.agileinfoways.com/wordpressplugins/wordpress/custom-add-meta-box/">Premium Version</a> with Adding Meta Box to Custom Post Type.

= Features =
**Wp Custom Meta Box** 
*Wp Custom Meta Box is helpful to create Meta Box and display on(Post, Page etc ).* 
*If data was added for Meta Box (Post, Page etc) from admin side then it will display on frontend for the same (Post, Page etc).*

= Wp Custom Meta Box Plugin will help you to Show Meta Box on Front-end by Following way :- =
* Create Custom Metabox from admin panel of Add New Wp Custom Metabox section.*	

= Create New Metabox by Adding =
	Meta Box Label
	Field Type from drop down (Textbox, Textarea, Drop Down, Check Box, Radio 
	Button)
	According to your selected Field Type
	Filled option below like Meta Field Type for(Textbox)
	Meta Field Title
	Meta Field Value for (Check Box, Radio Button,Drop Down)
	Meta Field Post Type
	Meta Field Required
	Meta Field Place Holder for (Textbox, Text Area)
	Meta Field Min Length for(Textbox)
	Meta Field Max Length for (Textbox) Status status as 'Available' OR Not 
	Available
*Admin will get that Meta Box on (Posts, Pages etc) as per selection of Meta Field Post Type.*
*Go to particular Post or Page and put data in meta box and update it.*
*Saved Meta Box data display on frontend of that Post or Page or any Post Type.*

== Installation ==
* Upload Wp Custom Meta Box Plugin Folder to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Create New Meta Box from Admin Panel
* Saved meta box with data on Post or Page or any Post types
* System will show meta data on frontend of that Post or Page or any Post types
